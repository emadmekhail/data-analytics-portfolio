# Abstractive Arabic Text Summarization

## Project Overview

This project focuses on building an Arabic text summarization system using Transformer-based NLP models.

---

## Tools Used

- Python
- Hugging Face Transformers
- PyTorch
- NLP
- REST API

---

## Main Steps

- Arabic text preprocessing
- Text cleaning and normalization
- Tokenization
- Fine-tuning Transformer models
- Model evaluation using ROUGE metrics
- API development for text summarization

---

## Models

- AraBERT
- mT5

---

## Project Files

- `NLTK-Summarizer-checkpoint.ipynb`  
  Main notebook containing the text summarization workflow.

- `requirements.txt`
