# Sales Dashboard

## Project Overview

This project presents an interactive dashboard designed to analyze sales performance, revenue, profit, regional performance, and product trends.

---

## Tools Used

- Power BI
- SQL
- Excel
- Data Visualization

---

## Key Features

- Revenue analysis
- Profit analysis
- Regional performance
- Product category analysis
- Date, region, and product filters
